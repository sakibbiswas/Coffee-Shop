# Coffee Shop Landing Page


## Overview
A modern, responsive landing page for a coffee shop built with React, Redux Toolkit, and Tailwind CSS.


## Features
- Hero, Menu (8 items), About, Testimonials, Contact with map
- Light/Dark theme toggle (persisted to localStorage)
- Favorites system (add/remove, persisted to localStorage)
- Smooth scroll navigation
- Responsive: mobile, tablet, desktop
## Live Link
[View Live Site](https://eclectic-beignet-3a99c7.netlify.app)
